# Cloud Drive

A simple cloud drive for file upload and download, deployed on Netlify.

## Features
- File upload
- File list display
- File download

## Deployment
This project is deployed on Netlify using Netlify Functions for backend processing.

## Local Development
```bash
# Install dependencies
npm install

# Start server
npm start
```

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Deployment: Netlify, Netlify Functions
